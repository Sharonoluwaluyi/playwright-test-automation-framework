# Playwright Test Automation Framework

## Overview
This project is an end-to-end test automation framework built using Playwright and TypeScript.
It demonstrates modern test automation practices including Page Object Model and CI/CD integration.

## Tech Stack
- Playwright
- TypeScript
- Node.js
- GitHub Actions

## Features
- End-to-end UI tests
- Page Object Model implementation
- CI pipeline for automated test execution
- Scalable and maintainable structure

## Project Structure
- tests/: Test specifications
- pages/: Page Object classes
- utils/: Test utilities and data
- .github/workflows/: CI configuration

## How to Run Tests
```bash
npm install
npx playwright test
